# Finance Dashboard 💰

A personal finance management dashboard built with React and Tailwind CSS, designed to help users track income, expenses, and overall financial health. It features a clean, responsive interface with dark mode support for comfortable use in any lighting condition.

Features
Dashboard Overview – Quick view of total income, total expenses, and current balance with summary cards.
Transaction Management – Add, view, filter, and sort all transactions.
Recent Transactions – Easily access the latest 5 transactions.
Transaction History – Search, filter by type/category/date, and sort transactions in a detailed table.
Charts – Visualize top expense categories to understand spending patterns.
Dark Mode – Toggle between light and dark themes for better accessibility and eye comfort.
Local Storage – All transactions and dark mode preferences persist between sessions.
Tech Stack
Frontend: React.js, React Router
Styling: Tailwind CSS
Icons: React Icons (FontAwesome)
Data Persistence: Local Storage
Installation
# Clone the repository
git clone https://github.com/Vartika0611/finance-dashboard.git

# Navigate into project directory
cd finance-dashboard

# Install dependencies
npm install

# Start the development server
npm start
Usage
Open the app in your browser at http://localhost:3000/.
Add transactions using the “Add Transaction” page.
Track income and expenses on the dashboard.
Filter and sort transactions in the “Transaction History” page.
Toggle dark mode using the switch in the navbar.
